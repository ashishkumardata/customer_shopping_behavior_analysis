# 🛒 Customer Shopping Behavior Analysis


## 📌 Project Overview

This project analyzes **customer shopping behavior** to uncover patterns, trends, and key drivers behind purchasing decisions.

It combines **Python, SQL, and Power BI** to transform raw data into meaningful business insights and interactive dashboards.

---

## 🎯 Objectives

* Understand customer purchase patterns
* Identify high-value customer segments
* Analyze factors influencing buying decisions
* Provide actionable business recommendations

---

## 🧰 Tools & Technologies

* 🐍 **Python (Pandas)** – Data cleaning & preprocessing
* 🧮 **SQL** – Data querying & analysis
* 📊 **Power BI** – Dashboard & visualization
* 💻 **GitHub** – Version control & project hosting

---

## 🔄 Project Workflow

1. Data Collection
2. Data Cleaning (Python)
3. Data Analysis (SQL)
4. Data Visualization (Power BI)
5. Insights & Business Recommendations

---

## 📊 Key Analysis Performed

* Customer segmentation analysis
* Purchase frequency & spending patterns
* Revenue contribution by customer groups
* Impact of discounts and promotions
* Trend analysis over time
